🚀 Lazy UI — Thư viện component React "copy-paste", đẹp mà nhẹ tênh

Chào cả nhà BIP VN 👋

Mình xin share với ae một dự án mình khá tâm đắc: **Lazy UI** — bộ component React + Tailwind, thiết kế đẹp, animation mượt, mà cài đặt cực gọn.

✨ Điểm hay:
• 50+ component chất lượng cao: text animation, background động, buttons, hiệu ứng, form, navigation, device mock...
• Cài thẳng vào project qua shadcn registry — không cần npm package, code về nằm gọn trong repo và sửa thoải mái.
• Tech stack mới nhất: React 19, Next.js 16, Tailwind CSS v4.
• Nhẹ, tree-shakeable, không "phình" bundle.
• Có sẵn blocks (pricing sections) responsive, dùng phát ăn ngay.

⚡ Cài đặt 1 dòng:
`npx shadcn@latest add https://2lazyui.com/r/matrix-grid.json`
