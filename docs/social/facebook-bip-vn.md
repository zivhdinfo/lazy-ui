Lazy UI — thư viện component React mình đang làm

Chào ae BIP VN,

Mình share với mọi người dự án này: Lazy UI. Điểm khác so với mấy thư viện thường gặp là component không cài qua npm package — source copy thẳng vào project của bạn, muốn sửa sao thì sửa, không bị khoá trong node_modules.

Hiện có hơn 50 component, tập trung vào mấy thứ tự làm cho mượt thì hơi mệt: text animation, background động (WebGL/Three.js), hiệu ứng con trỏ, các kiểu transition... Ngoài ra có sẵn vài block pricing responsive để ráp nhanh.

Web: https://2lazyui.com

Ae nào làm frontend ghé thử xem sao, thấy ổn thì cho mình xin ý kiến. Có bug hay góp ý gì cứ nói thẳng nhé.
